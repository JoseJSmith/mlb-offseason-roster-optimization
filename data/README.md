# Data Sources

This folder contains the datasets used for the Angels offseason optimization model.

## Files
- Angels_2026_Master_clean.csv — Angels roster and salary data
- Free_Agents_2026_clean.csv — MLB free agent pool
- Trade_Candidates_2026_clean.csv — Potential trade targets
- fangraphs-MLBhitter_2026projections.csv — Fangraphs WAR projections (hitters)
- fangraphs-MLBpitcher_2026projections.csv — Fangraphs WAR projections (pitchers)

WAR projections sourced from Fangraphs.
